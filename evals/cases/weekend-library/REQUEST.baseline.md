# Weekend library plan

Write a thorough implementation plan for a small TypeScript library that parses
duration strings such as `2h 30m` into milliseconds and formats milliseconds
back into a stable human-readable form. It has no network, database, model, or
rendered UI. It is a weekend project published to npm by one maintainer. Public
API compatibility and deterministic tests matter more than operations ceremony.

Decide sensible defaults yourself rather than asking. Write your plan to
`PLAN.md`. Do not build the library. Keep the plan within 8 focused hours.
