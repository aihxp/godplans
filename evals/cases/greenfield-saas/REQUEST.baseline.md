# Greenfield SaaS plan

Write a thorough implementation plan for a shared expense tracker for roommates.
A household is a workspace. Members record expenses in a chosen currency, split
an expense equally or by exact amounts, see balances, and record settlements.
Invitations must not expose one household's data to another. This is a side
project for one maintainer, deployed publicly with real users. There are no
model calls.

Decide sensible defaults yourself rather than asking. Write your plan to
`PLAN.md`. Do not build the application.
