# Greenfield SaaS evaluation

Use godplans to plan a shared expense tracker for roommates. A household is a
workspace. Members record expenses in a chosen currency, split an expense
equally or by exact amounts, see balances, and record settlements. Invitations
must not expose one household's data to another. This is a side project for one
maintainer, deployed publicly with real users. There are no model calls.

Take recommended defaults when the skill asks questions. Produce the complete
godplans artifact set under `.godplans/`, including `PLAN.mdx` and its
executable validator companion. Do not build the application.
